# Parallax Podcast (Vite + React + Tailwind)

This is a minimal scaffold for a mobile-first Parallax Podcast app built with Vite, React and Tailwind.

How to run (Windows cmd):

1. Install dependencies

```cmd
cd parallax-app
npm install
```

2. Start dev server

```cmd
npm run dev
```
